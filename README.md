# Simple money manager

## Description

A simple app for working with your finances.

This app provides:

1. Signing in/out/up
2. Creating and deleting accounts
3. Creating and deleting transactions 

## Technologies:

- HTML
- CSS
- JS
- Webpack
- ESLint
- Babel
- Express

## Initialising

1. git clone
2. npm install
3. npm run build
4. npm start
